# Getting Started

Three steps and you are in. Each step takes under a minute.

1. [Connect a wallet](./wallet) on the LiteForge chain (4441).
2. [Claim the faucet](./faucet) for 0.01 zkLTC and 10 starter points.
3. [Register a .lit name](./lit-name) so other users can find you.

## Network details

| Field | Value |
| --- | --- |
| Chain | LiteForge |
| Chain ID | `4441` |
| Native token | zkLTC |
| RPC URL | `https://liteforge.rpc.caldera.xyz/http` |
| Explorer | `https://liteforge.explorer.caldera.xyz` |

The LitDEX dApp prompts to add the network the first time you sign a transaction. If you prefer to add it manually, open MetaMask → Settings → Networks → Add network and paste the values above.

## Recommended path

After your first faucet claim:

- **Try a swap** so the router cache warms up and your wallet shows balances.
- **Mint a Genesis NFT** if you have enough points — it unlocks daily passive rewards.
- **Send a message** in the [Hub](/hub/) global feed to start your on-chain reputation.
- **Complete socials** — follow @LitDeXApp on X and join the Telegram for free points.

> A wallet that has held a LitDEX NFT and a `.lit` domain unlocks the faucet, daily rewards, and points multipliers.
